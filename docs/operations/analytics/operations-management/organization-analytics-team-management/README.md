## Quick Reference

### Role Hierarchy

| Role | Permissions |
|------|-------------|
| **Owner** | Full access, billing, delete company |
| **Admin** | Team management, settings, campaigns |
| **Member** | Create campaigns, view analytics |

### Team Health Metrics

| Metric | Description | Target |
|--------|-------------|--------|
| Team Utilization | Active / total invited | > 80% |
| Invitation Acceptance | Accepted / sent | > 90% |
| Onboarding Completion | Completed setup | > 95% |
