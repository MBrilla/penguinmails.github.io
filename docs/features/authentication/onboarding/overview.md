# Onboarding Experience Overview

Get new users productive in under 10 minutes with guided setup, interactive tutorials, and contextual help.

## Overview

The Onboarding Experience provides a structured, interactive journey for new users from signup to their first successful email campaign, reducing time-to-value and improving activation rates.

## Document Structure

This feature is organized into focused modules:

1. **[User Flow](/docs/features/authentication/onboarding/user-flow)** - Step-by-step onboarding journey
2. **[Advanced Features](/docs/features/authentication/onboarding/advanced-features)** - Checklists, tooltips, and milestones
3. **[Technical Implementation](/docs/features/authentication/onboarding/implementation)** - Database schema and services

## Key Components

| Component | Purpose |
|-----------|---------|
| Guided Setup Wizard | Step-by-step infrastructure configuration |
| Interactive Checklist | Track progress toward first campaign |
| Contextual Tooltips | In-app guidance and feature discovery |
| Video Tutorials | Quick how-to videos at key moments |
| Progress Tracking | Visual progress indicators |
| Achievement Milestones | Celebrate user progress |

## New User Journey

```text
1. Sign Up → 2. Workspace Setup → 3. Domain Configuration →
4. Payment → 5. First Email Account → 6. First Campaign → 7. Success!
```

**Time to Complete**: 8-10 minutes

## Journey Steps Summary

| Step | Duration | Description |
|------|----------|-------------|
| Sign Up | 1 min | Email, password, company name |
| Create Workspace | 1 min | Organize campaigns by client/project |
| Add Domain | 2 min | Sending domain configuration |
| Verify Domain | 2 min | DNS record verification |
| Set Up Payment | 1 min | Plan selection and trial start |
| Provision Infrastructure | 2-3 min | VPS and SMTP setup |
| Create Email Account | 1 min | First sending account |
| Success | 30 sec | Dashboard and next steps |
